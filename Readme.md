# Custom Unix Shell

A minimal Unix-like shell implemented in C. This project focuses on understanding how command-line interpreters work at a low level, including process creation, command execution, and environment handling.

## Features

- Execute system commands using fork and execve
- Argument parsing
- Environment variable handling
- Custom string utility functions
- Basic error handling



## Compilation & Usage

Compile:
```bash
make
```
Run:
```bash
./shell
```

## Learning Outcomes

- Process management (fork, execve, wait)
- Memory management and error handling
- Understanding how shells execute commands
- Building core utilities from scratch

## Notes

- This is a learning project and not a full-featured shell
- Behavior may differ from standard shells like bash
